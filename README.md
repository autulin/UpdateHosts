Hosts一键更新 0.1Beta

更新Hosts是最方便快捷的科学上网方法，并且还可以去除广告，缺点是对部分CDN无效（如可以上Google Play但不能下载东西，能打开Youtube但不一定能看视屏），另外有一定的时效性，需定期更新，这是关键，故造此轮，可以更方便的更新各种Hosts源，平时用用Google什么的毫无压力
